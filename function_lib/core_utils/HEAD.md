# Functional Group: core_utils

## Overview
This directory serves as a standard template for how compiled, stable functional groups should be documented in the `function_lib` system. The `core_utils` group acts as an example containing general-purpose utility operations.

## Contained Function Documents
- `python_test.md`: A template function demonstrating how to use bash to invoke Python and print "hello world".
- `example_parser.py`: A robust script for parsing text inputs into standardized formats.
- `network_helper.js`: Utilities for making resilient HTTP requests with built-in retries.
- `architecture_guide.md`: Documents the overarching design pattern of the utility scripts in this group.

## Contained Skills
- **Text Formatter**: Cleans up and normalizes raw text inputs from the user.
- **Request Engine**: Handles outbound API queries safely.
