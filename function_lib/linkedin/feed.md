# Summarize LinkedIn Feed

## Purpose
Fetches and reads the latest top N posts from the user's authentic home feed, skipping ads.

## Action Execution
```bash
lk feed -n 10
```
*(Adjust the integer parameter `-n 10` depending on how many posts the user wants to ingest or summarize).*

## Success Verification
The feed returns chronological posts or a structured text report without network connection drops or cookie errors.
