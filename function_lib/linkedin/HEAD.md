# LinkedIn Capability Set 💼

This module provides an independent LinkedIn CLI agent powered by local session cookies, enabling deep profile research, messaging, and feed parsing.

## Available Functions

- `linkedin/whoami.md`: Display current authenticated LinkedIn profile details.
- `linkedin/search.md`: Search for LinkedIn people/candidates by precise query string.
- `linkedin/profile.md`: Fetch a detailed, comprehensive JSON summary of a specific user's public ID.
- `linkedin/feed.md`: Summarize and extract the top recent posts from the home timeline.
- `linkedin/messages.md`: Quick peek at the latest active LinkedIn inbox conversations.
- `linkedin/send.md`: Send a direct private message to a connection by their exact name.
