# LLM Chat (Fallback)

A universal fallback skill that handles any task by directly conversing with the cloud LLM.

## When to Use

This skill is automatically used when:
- No specialized skill matches the user's request
- The task requires general knowledge or reasoning  
- A creative, analytical, or conversational response is needed

## Capabilities

- Answer questions on any topic
- Summarize text or documents
- Generate creative content (emails, messages, stories)
- Analyze data or code
- Translate between languages
- Provide step-by-step instructions
- General reasoning and problem-solving

## Usage

Simply describe your task or question in natural language. The LLM will process it and return a response.

**Example:**
```
User: Summarize the key points of this article about AI safety
User: Write a professional email declining a meeting invitation
User: Explain the difference between TCP and UDP
```

## Notes

- No external tools, APIs, or commands are needed
- Responses are generated by the cloud LLM model
- Best for tasks that don't require local system access or specific tool execution
