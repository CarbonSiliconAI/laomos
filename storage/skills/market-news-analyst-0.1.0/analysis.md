# Market News Analyst Skill - Analysis Criteria Document

## 1. Performance Metrics to Track

### 1.1 Coverage Metrics

| Metric | Target | Measurement Method | Acceptable Range |
|--------|--------|-------------------|------------------|
| **News Events Identified** | 8-12 significant events per 10-day period | Count of events with Impact Score >5 | 6-15 events |
| **Source Diversity** | ≥3 different Tier 1 news sources per report | Count unique sources cited | 3+ sources |
| **Asset Class Coverage** | Analysis covers ≥4 asset classes | Equities, Bonds, Commodities, Currencies present | 4+ asset classes |
| **Search Query Execution** | 100% of planned search queries executed | Count executed vs planned searches | 95-100% |
| **Data Freshness** | 100% of analyzed news within 10-day window | Verify all event dates fall within period | 100% compliance |
| **Sector Analysis Depth** | ≥5 sectors analyzed when applicable | Count sectors with specific impact assessment | 5+ sectors |

### 1.2 Quality Metrics

| Metric | Target | Measurement Method | Acceptable Range |
|--------|--------|-------------------|------------------|
| **Impact Score Accuracy** | Correlation >0.85 with actual market moves | Compare predicted vs actual price movements | >0.80 correlation |
| **Attribution Confidence** | ≥80% of major moves attributed with confidence | Percentage of moves with clear causal link | 70-100% |
| **Pattern Match Accuracy** | ≥75% of reactions match historical patterns | Compare observed vs expected reactions | 65-85% accuracy |
| **Quantification Completeness** | 100% of major price moves include specific % | Every move includes precise number | 95-100% |
| **Source Credibility** | 100% of primary claims from Tier 1-2 sources | Verify source tier for each major claim | 100% Tier 1-2 |
| **Anomaly Identification** | ≥1 anomaly identified per report | Count significant deviations from patterns | 1-3 per report |

### 1.3 Analysis Depth Metrics

| Metric | Target | Measurement Method | Acceptable Range |
|--------|--------|-------------------|------------------|
| **Multi-Asset Reaction Detail** | All 5 asset classes analyzed for top 3 events | Equities, Bonds, Commodities, Currencies, Derivatives | 4+ asset classes |
| **Geopolitical-Commodity Correlation** | 100% of geo events include commodity analysis | Verify commodity section for each geo event | 100% coverage |
| **Causation vs Correlation Rigor** | Explicit distinction in ≥80% of multi-event analysis | Count events where causation/correlation separated | 75-100% |
| **Forward-Looking Content** | ≥3 forward-looking implications per report | Count implications for future markets | 3+ implications |
| **Sector Rotation Analysis** | Growth vs Value, Cyclical vs Defensive analyzed | Both rotation dimensions present | 2/2 present |
| **Risk Scenario Identification** | ≥3 specific risk scenarios identified | Count quantified risks with probabilities | 3-5 scenarios |

### 1.4 Output Quality Metrics

| Metric | Target | Measurement Method | Acceptable Range |
|--------|--------|-------------------|------------------|
| **Report Completeness** | All 6 sections present and substantive | Check for Executive Summary, Rankings, Detailed Analysis, Synthesis, Commodity Deep Dive, Forward Implications | 6/6 sections |
| **Impact Score Calculation** | All top 10 events include full calculation | Verify formula shown: (Price × Breadth) × Forward | 100% of top 10 |
| **Table Accuracy** | Impact Rankings table error-free | No calculation errors, proper ranking order | 100% accuracy |
| **Citation Completeness** | ≥95% of major claims include source | Verify source citation for facts | 90-100% |
| **Markdown Formatting** | Valid Markdown syntax, proper structure | No broken links, proper heading hierarchy | 100% valid |
